# Vector Graphics Accelerator

## What is this
This is a class project to make a simple GPU with an FPGA.

## How do I run it
1. Get an iCEBreaker with the HDMI PMOD
2. Install Amaranth and the IceStorm toolchain
3. Run `./main.py --flash`
